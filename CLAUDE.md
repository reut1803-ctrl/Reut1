# הנחיות עבודה לפרויקט (חשוב מאוד!)

> קובץ זה נקרא אוטומטית בכל פעם. הוא מגדיר איך לעבוד מול המשתמשת.

## על המשתמשת (ראות)
- **אין לה רקע טכנולוגי בכלל.** אסור להניח שום ידע טכני.
- **אנגלית מבלבלת אותה.** הכול חייב להיות בעברית.

## כללי זהב — לעבוד תמיד ככה
1. **הכול בעברית** — כל הטקסטים באתר (כפתורים, כותרות, הודעות), כל ההסברים
   בצ'אט, וכל קובץ או מדריך שנכתב למשתמשת. בלי אנגלית מול המשתמשת.
2. **כיוון מימין לשמאל (RTL)** בכל ממשק המשתמש.
3. **אני עושה את כל החלקים הטכניים הקשים בעצמי.** לא מבקשים ממנה
   להריץ פקודות מסובכות או להבין קוד.
4. **תיווך צעד-אחר-צעד.** בסוף כל תשובה אומרים לה במשפט פשוט אחד
   מה הדבר הבא שעליה לעשות (אם בכלל). בלי ז'רגון.
5. **להסביר בפשטות.** אם חייבים מונח טכני — מסבירים אותו במילים פשוטות.
6. **לבדוק לבד שהכול עובד** (build, בדיקות) לפני שמדווחים שסיימנו.

## על הפרויקט
פלטפורמה לשידוכים וניהול ראיונות עבור רכזת קהילה.
- מסד נתונים: Supabase (PostgreSQL) — קבצי SQL בתיקייה `supabase/`.
- צד לקוח: React + Vite — בתיקייה `src/`.
- "מצב דמו": אם אין חיבור למסד נתונים אמיתי, האתר עובד עם נתוני דוגמה
  בעברית כדי שאפשר יהיה לראות אותו מיד.
