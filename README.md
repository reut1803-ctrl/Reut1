# מאגר שידוכים - מערכת CRM

מערכת ניהול מאגר שידוכים לצוות שדכניות. בנויה ב-Next.js, Tailwind CSS ו-Zustand, מותאמת לנייד (Mobile-First), בעברית מלאה מיושרת לימין (RTL).

## שלב א' (המצב הנוכחי)

בשלב הזה נבנה השלד המלא של המערכת עם **נתוני דמה** לצורך תצוגה - עדיין אין חיבור למסד נתונים אמיתי. זה יגיע בשלב הבא (Supabase).

מה יש כרגע:

- **דף נחיתה** עם הרשמה / כניסה / כניסה עם גוגל וקרוסלת "סיפורי הצלחה".
- **שלד ניווט מלא**: סרגל עליון (ברכת שלום, התראות, הגדרות, ריענון, התנתקות), החלפה בין מאגר בנים למאגר בנות, סרגל תחתון צף (מועדפים / התאמות / פרופילים), וכפתורים צפים (נגישות + וואטסאפ).
- **עמוד הפיד**: טאבים "הצעות חדשות" / "הצעות קודמות", סינון חכם (גיל, גובה, רמת תורניות, אזור, חיפוש חופשי), כרטיסיות מועמדים עם כפתורי פעולה, ושמירת מיקום גלילה כשחוזרים לעמוד.
- **שאלון ההתאמות** בן 7 השלבים, עם מסך תוצאות ("X התאמות גבוהות מעל 70%").
- **מסך מועדפים**.
- **3 רמות הרשאה להדגמה** (ניתן להחליף בין צופה / צוות / מנהלת דרך הגדרות > מצב תצוגה להדגמה) - כדי לראות איך האזור הפנימי (הקלטת שמע, הערות, סטטוס במשפך) מוצג רק לצוות ולמנהלת.

## איך מריצים את זה אצלך במחשב

לא צריך לדעת לתכנת בשביל זה - רק לעקוב אחרי השלבים:

1. **להתקין Node.js** (פעם אחת בלבד) - נכנסים לאתר https://nodejs.org ומורידים את הגרסה המומלצת (LTS), מתקינים כמו כל תוכנה רגילה.
2. **להוריד את הקוד למחשב** - אם עובדים דרך GitHub, לוחצים על הכפתור הירוק "Code" ואז "Download ZIP", ומחלצים את הקובץ לתיקייה כלשהי.
3. **לפתוח את התיקייה בטרמינל**:
   - ב-Windows: פותחים את התיקייה בסייר הקבצים, לוחצים על שורת הכתובת למעלה, מקלידים `cmd` ולוחצים Enter.
   - ב-Mac: פותחים את אפליקציית Terminal, מקלידים `cd ` (עם רווח) וגוררים את התיקייה לתוך החלון, ואז Enter.
4. **להתקין את כל מה שהמערכת צריכה** - מקלידים בטרמינל:
   ```bash
   npm install
   ```
   (פעולה חד פעמית, לוקחת דקה-שתיים).
5. **להריץ את המערכת**:
   ```bash
   npm run dev
   ```
6. **לפתוח בדפדפן** את הכתובת: http://localhost:3000

זהו - האתר רץ אצלך במחשב בדיוק כפי שהוא ייראה באינטרנט. כל שינוי שאני אעשה בקוד יופיע אוטומטית בדפדפן.

## פרסום האתר לאינטרנט (Vercel)

1. נכנסים ל-https://vercel.com ומתחברים עם חשבון GitHub.
2. לוחצים **Add New… → Project**.
3. בוחרים את המאגר הזה (Reut1) ולוחצים **Import**.
4. משאירים את כל ההגדרות כמו שהן ולוחצים **Deploy**.
5. תוך דקה מתקבל קישור חי לאתר.

## מה בשלבים הבאים

- חיבור אמיתי למסד נתונים מאובטח (Supabase) עם הרשאות אמיתיות במקום המתג להדגמה.
- מסך התחברות/הרשמה אמיתי (כולל כניסה עם גוגל בפועל).
- דאשבורד ניהול למנהלת (נתונים בזמן אמת, משימות ולו"ז, משפך התאמות מלא).
- עריכת כרטיסיות והוספת מועמדים חדשים על ידי המנהלת.
